Videos del modal "Ver cómo" (sección Servicios de la landing).

Dejá acá los .mp4 con EXACTAMENTE estos nombres (cada uno corresponde a un servicio):

  web-lista.mp4           -> "Tu web lista"
  que-te-encuentren.mp4   -> "Que te encuentren"
  consultas-ordenadas.mp4 -> "Consultas ordenadas"
  mejora-cada-mes.mp4     -> "Mejora cada mes"

- El servicio que NO tenga su archivo muestra el cartel "Próximamente" como respaldo.
- Recomendado: MP4 (H.264 + AAC), vertical u horizontal, peso liviano (idealmente < 8 MB).
- Si querés otros nombres o más videos, avisá y ajusto el mapeo en src/App.tsx (serviceVideos).
